# 🃏 Blackjack Game (Python)
A simple command-line Blackjack game built using Python.
This project was created as part of the **100 Days of Code™: The Complete Python Pro Bootcamp
 by Angela Yu** on Udemy.
---
## 🚀 Features
* Random card dealing system
* Blackjack detection (Ace + 10-value card)
* Dynamic Ace value adjustment (11 → 1)
* Computer opponent logic (draws cards until score reaches 17)
* Interactive user input (hit or stand)
* Replay option to play multiple rounds
---
## 🛠️ Technologies Used
* Python 3
* Built-in `random` module
---

## ▶️ How to Run
1. Clone the repository:
```bash
git clone https://github.com/your-username/blackjack-game.git
```
2. Navigate to the project folder:
```bash
cd blackjack-game
```
3. Run the game:
```bash
python blackjack.py
```
---
## 🎮 How to Play
* You and the computer are dealt two cards each
* Try to get as close to 21 as possible without going over
* Type `'y'` to draw another card (Hit)
* Type `'n'` to pass (Stand)
* The computer will automatically draw until it reaches at least 17
